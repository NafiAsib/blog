### Installation

- [Create a Next.js app](https://nextjs.org/docs/pages/api-reference/create-next-app)

```bash
pnpm create next-app
```

- Install `contentlayer`

```bash
pnpm install next-contentlayer
```

- Then wrap your next configuration object in the withContentlayer utility

```jsx
// next.config.js
import { withContentlayer } from "next-contentlayer";
export default withContentlayer({});
```
